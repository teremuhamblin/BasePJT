tests/

Ce dossier contient les tests unitaires et fonctionnels du projet BasePJT.

Contenu

- test_core.py — Tests de base pour vérifier le cœur du projet.
- test_config.py — Tests liés aux fichiers de configuration.

Objectifs

- Vérifier la stabilité du projet.
- Détecter les régressions.
- Garantir une base solide avant chaque release.

Exécution

Depuis la racine du projet :

```bash
pytest
```

Pour plus de détails :

```bash
pytest -vv
```
`
