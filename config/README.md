# config/

Ce dossier contient toutes les configurations essentielles du projet **BasePJT**.

## Contenu

- **settings.yml** — Configuration principale du projet (logging, réseau, version, mode).
- **environment.conf** — Variables d’environnement utilisées par les scripts et l’application.

## Objectif

Centraliser les paramètres afin de garantir :
- une structure propre,
- une maintenance simplifiée,
- une configuration facilement modifiable,
- une cohérence militaire dans l’organisation du projet.
